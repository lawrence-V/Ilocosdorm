# IlocosDorm
IlocosDorm is a community-driven platform that helps students, professionals, and travelers find verified dormitories, boarding houses, and affordable rentals across Ilocos. The platform provides an easy way to search available accommodations, compare prices, explore amenities, and view locations on an interactive map—all in one place.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

Start editing the app by modifying `app/page.tsx`.
